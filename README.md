dotfiles
========

## installation

1. clone to your home directory  
````
    git clone git@github.com:uchidas/dotfiles.git
````
2. delete or move local dotfiles  
3. exectue setup.sh  
````
    sh setup.sh
````
4. source .zshrc  
````
    source .zshrc
````
